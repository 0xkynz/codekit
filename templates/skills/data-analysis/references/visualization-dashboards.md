## Visualization Libraries

### Static & Publication
- [Matplotlib](https://matplotlib.org/stable/contents.html) - Comprehensive static, animated, interactive viz
- [Seaborn](https://seaborn.pydata.org/) - Statistical data visualization on Matplotlib
- [Plotnine](https://plotnine.readthedocs.io/en/stable/) - Grammar of graphics (ggplot2-style)
- [Pygal](https://pygal.org/en/stable/) - SVG charting library
- [PyPalettes](https://github.com/JosephBARBIERDARNAL/pypalettes) - 2500+ color maps

### Interactive
- [Plotly](https://plotly.com/python/) - Interactive plots and dashboards
- [Altair](https://github.com/vega/altair) - Declarative statistical visualization
- [Bokeh](https://docs.bokeh.org/en/latest/) - Interactive viz for web browsers
- [HoloViews](https://holoviews.org/) - Build complex visualizations easily
- [Bqplot](https://github.com/bqplot/bqplot) - Plotting for IPython/Jupyter
- [Pandas-bokeh](https://github.com/PatrikHlobil/Pandas-Bokeh) - Bokeh plotting backend for Pandas

### Geographic & Spatial
- [Folium](https://python-visualization.github.io/folium/) - Data on interactive maps
- [GeoPandas](https://geopandas.org/en/stable/) - Geospatial data extension for Pandas
- [Contextily](https://github.com/geopandas/contextily) - Background basemaps for GeoPandas
- [OSMnx](https://github.com/gboeing/osmnx) - OpenStreetMap network analysis
- [QGIS](https://github.com/qgis/QGIS) - Open source GIS

### Large Data & 3D
- [Datashader](https://github.com/holoviz/datashader) - Render large data quickly
- [Deck.gl](https://github.com/visgl/deck.gl) - WebGL-powered large dataset viz
- [VisPy](https://github.com/vispy/vispy) - High-performance 2D/3D viz (OpenGL)
- [Glumpy](https://github.com/glumpy/glumpy) - Scientific viz based on OpenGL

### Web & No-Code
- [Apache ECharts](https://github.com/apache/echarts) - Interactive charting for browsers
- [Flourish](https://flourish.studio/) - No-code interactive visualizations
